Все реальные данные, связанные с персональными данными были заменены на вымышленные, совпадения случайны


- `generate_jira_data` - скрипт для генерации тестовых заявок из JIRA, создает 4 файла:
- `jira_aggregated_data.csv` - агрегированные данные, минимально необходимые для проведения A/B-теста;
- `jira_daily_stats.csv` - ежедневная статистика;
- `jira_full_export.csv` - экспорт заявок, содержащих все данные из заявки, из JIRA;
- `jira_simple_export.csv` - экспорт заявок из JIRA с необходимыми для исследования полями.
